                            Aptitude On-Line Hjälp

  Använd pil-knapparna och PageUp/PageDown för att se denna hjälp, tryck 
Return eller Enter för att lämna den.

  Följande knappar är tillgängliga som standard i aptitude's informations-
vyer.  Andra knappar är och tillgängliga, se README för en lista av dessa.

  "q":          Avsluta.
  "?":          Visa denna skärm.

  "f":		Töm listan av "nya" paket ('f' för "forget"(glömma)).

  Up eller "k": Flytta texten upp.    
  Down eller "j": Flytta texten ned. 
  "^":          Flytta valet till föräldern för nuvarande sak.
  Enter:        Expandera eller minska en grupp av paket.
  "[":          Expandera en grupp av paket och alla dess undergrupper.
  "]":          Förminska en grupp och alla dess undergrupper.

  Enter:        Se information om ett paket.
  "C":          Visa ändringsloggen för ett paket.
  "+":          Installera eller uppgradera ett paket eller ta bort 
                status för tillbakahållning.
  "-":          Ta bort ett paket.
  "=":          Håll kvar ett paket i nuvarande version för att undvika 
                uppgradering.
  ":":          Håll kvar ett paket i nuvarande version. Kommer inte att 
                undvika framtida uppgraderingar.
  "_":          Sätt att ett paket och all dess konfiguration tas bort.
  "L":          Sätt att ett paket ska installeras om.
  "M":		Markera ett paket som automatiskt installerad.
                Automatiskt installerade paket tas bort om inga manuellt 
                installerade paket kräver dom.
  "m":		Markera ett paket som manuellt installerad.
  "F":		Förbjud ett paket från att bli automatiskt uppgraderad 
                till en viss version; nyare versioner kommer automatiskt 
                bli installerade.

  "u":          Uppdatera listan av tillgängliga paket.
  "U":		Markera alla uppgraderingsbara paket för uppgradering.
  "g":          Genomför alla tänkta installationer, borttagningar och 
                uppgraderingar.

  Control-u:    Ångra den senaste åtgärdet eller grupp av åtgärder.
  "/":          Genomför en sökning.  (på namnet av paketet som standard;
                se README/User's Manual för mer information).
  "\":          Upprepa senaste sökningen.
  "b":          Sök efter nästa brutna paket.

  "v":          Visa de tillgängliga versionerna av ett paket.
  "d":          Visa beroenden för ett paket.
  "r":          Visa paketen som är beroende av ett angivet paket.

  "D":          Visa eller göm rutan med paketinformation.
  "a":		Skrolla upp i rutan med paketinformation.
  "z":		Skrolla ned i rutan med paketinformation.
  "i":          Bläddra igenom paketinformationsvyerna.

  Flaggan "status" som visas för ett paket har följande möjliga värden:

   v - virtuell
   B - bruten
   u - "uppackad"
   C - halvt-konfigured
   H - halvt-installerad
   c - borttagen, men konfiguration finns kvar (alltså, inte rensad)
   i - installerad
   E - internt fel (ska inte hända)

  Flaggan "åtgärd" som visas för ett paket har följande möjliga värden:

   h - håll kvar
   p - rensa
   d - radera (ta bort)
   B - bruten
   i - installera
   r - installera om
   u - uppgrada
   F - den tillgängliga uppgradering har förbjudits via "F"
